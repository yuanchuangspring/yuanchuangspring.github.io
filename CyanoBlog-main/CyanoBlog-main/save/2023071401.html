写IT类博客，最重要的一点就是将代码呈现出来，必须要实现代码高亮等基本功能，且必须兼容不同的编程语言。现在比较主流的第三方包就是prism，它支持自选语言，在线定制打包，因此更加轻巧简易。

首先我们进到它的官网：https://prismjs.com/download.html

可以看到，prism提供了大部分主流语言以供选择，选好之后，点击页面下方的download js 和 download css ，下载好两个文件之后在html中引入即可

在html中，我们可以用如下方式将代码嵌入

<pre>
    <code class='language-js'> 
        console.log("Hello World")
    </code>
</pre>

注：在langua-的后面要加上所用的语言
